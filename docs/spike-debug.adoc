.*Note: the meaning of the abbreviations in the table*
[%collapsible]
====
****
*For instruction operands:*

I:: Integer register
C:: Capability register
S:: Used as sign-extended immediate
Z:: Used as zero-extended immediate
-:: Not used

*For immediates:*

S:: Sign-extended
Z:: Zero-extended
-:: Not used

*For worlds:*

N:: Normal world
S:: Secure world
*:: Either world
****
